ESP32-S3-Touch-LCD-4 新风机外设引脚核对表
==========================================

一、ESP32 直连（信号脚 + I2C 总线）
------------------------------------------
GPIO   板载原用途        本方案          结论
7      I2C SCL          外挂 I2C SCL    ✓ 与触摸/RTC 同 bus_a
15     I2C SDA          外挂 I2C SDA    ✓
4      SD MISO          送风 CLK PWM    ✓ 不用 TF 卡
6      CAN TX           送风 FG         ✓ 不用 CAN
0      CAN 等           排风 CLK PWM    ✓ Strapping，上电前勿强拉低
16     触摸 INT         排风 FG         ✓ 固件 pullup
43     RS485 RX         PMS5003 TX→RX   ✓ 不用 RS485
44     RS485 TX         PMS5003 RX←TX   ✓

不可用/已占用：GPIO1/2/3/5/8–14/17–21/38–42/48 等（LCD/RGB/USB/触摸）


二、I2C 设备地址（bus_a = GPIO15/7，建议 100kHz）
-------------------------------------------------
0x20   V3 板载 TCA9554（勿占）→ 外挂 MCP 用 0x21
0x24   V4 板载 CH32V003
0x44   SHT31 室内
0x45   SHT31 室外
0x51   V4 RTC PCF8563
0x5D   触摸 GT911
0x61   SCD30
0x21   MCP23017（推荐默认；V4 可改 0x20）


三、MCP23017（16 脚，全部占用规划）
-----------------------------------
Pin   方向    功能
0–3   出      进风 35BYJ ULN IN1–IN4
4–7   出      排风 ULN
8–11  出      混风 ULN
12–14 预留（接关限位时用于 limit 模式，见 actuators.yaml）
15    —       预留

结论：12 步进占用；无限位时 12–15 均可不接

六、风阀归零（无三路关限位时）
--------------------------------
默认 stall：关向顶机械硬挡块 → 超时 → 坐标置 0（需关死位挡块）
可选 manual：上电不自动动，点 HA「风阀确认已关闭」置 0
可选 limit：MCP 12/13/14 接关限位，valve_home_mode: limit

关位步数 0；开位进/排/混各自独立（actuators.yaml）
上电/断电：stall 自动寻零；手动「风阀重新归零」可再跑一遍


四、Nidec 48F704P400 风机（6 针座，常用 4 线）
----------------------------------------------
板端丝印（上→下）  线色    接法
BRK               —       不接
FG                橙      ESP FG + 3.3V 上拉（送风 GPIO6 / 排风 GPIO16）
CLK               蓝      ESP LEDC PWM 25kHz（送风 GPIO4 / 排风 GPIO0）
5V                —       不接
GND               黑      24V- 与 ESP GND 共地
24V               红      24V 电源正

固件 fan.yaml：PWM 直连 CLK，FG 接 pulse_counter。


五、电源与驱动
--------------
送风/排风 ECM：各 24V（约 ≤1.8 A/台），与 ESP 共地
35BYJ-46 + ULN2003 ×3：电机侧 12V≥2A，逻辑共地
MCP 只驱动 ULN 输入端，不承受电机电流 ✓
PMS5003：5V（勿从 ESP 3.3V 供电）
SHT31 / SCD30：3.3V（与 ESP 同总线）


六、冲突检查
------------
[✓] 不用 SD → GPIO4 送风 CLK
[✓] 不用 CAN → GPIO6 送风 FG；GPIO0 排风 CLK
[✓] 不用 RS485 → GPIO43/44 可用
[✓] 三路步进不占 ESP GPIO
[!] 步进经 I2C 扩展，转速已限制 140 steps/s
[!] 与 LVGL/触摸同总线，勿设 800kHz I2C
