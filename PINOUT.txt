Guition ESP32-S3-4848S040 新风机外设引脚核对表
==================================================
参考：https://devices.esphome.io/devices/guition-esp32-s3-4848s040/
      NorthernMan54/ESP32-4848S040 GPIO 表

一、ESP32 直连（信号脚 + I2C 总线）
------------------------------------------
GPIO   板载原用途              本方案              结论
19     触摸 I2C SDA / USB-     外挂 I2C SDA        ✓ bus_a，接 SHT/SCD/MCP
45     触摸 I2C SCL            外挂 I2C SCL        ✓
43     UART0 TX（板载座）      PMS5003 RX←TX       ✓
44     UART0 RX（板载座）      PMS5003 TX→RX       ✓
35     扩展（未接屏 RGB）      送风 CLK PWM        ✓ 须从屏排线侧引出/飞线
36     扩展                    送风 FG             ✓
37     扩展                    排风 CLK PWM        ✓
33     扩展                    排风 FG             ✓

勿用（被 480×480 RGB 屏占用）：
  GPIO0,3–18,20,21,39,46,47,48 等 — 含原 Waveshare 方案的 4/6/0/16

板载占用勿挪作它用：
  GPIO38 背光   GPIO40/1/2 继电器（部分批次有三路继电器）


二、I2C 设备地址（bus_a = GPIO19/45，建议 100kHz）
-------------------------------------------------
0x20   MCP23017（默认；扫描确认）
0x44   SHT31 室内
0x45   SHT31 室外
0x5D   触摸 GT911（板载，勿与 MCP 冲突）
0x61   SCD30


三、MCP23017（16 脚，全部占用规划）
-----------------------------------
Pin   方向    功能
0–3   出      进风 35BYJ ULN IN1–IN4
4–7   出      排风 ULN
8–11  出      混风 ULN
12–14 预留（接关限位时用于 limit 模式）
15    —       预留


四、Nidec 48F704P400 风机（6 针座，常用 4 线）
----------------------------------------------
板端丝印（上→下）  线色    接法
BRK               —       不接
FG                橙      ESP FG + 3.3V 上拉（送风 GPIO36 / 排风 GPIO33）
CLK               蓝      ESP LEDC PWM 25kHz（送风 GPIO35 / 排风 GPIO37）
5V                —       不接
GND               黑      24V- 与 ESP GND 共地
24V               红      24V 电源正

固件 fan.yaml：PWM 直连 CLK，FG 接 pulse_counter。


五、电源与驱动
--------------
送风/排风 ECM：各 24V，与 ESP 共地
35BYJ-46 + ULN2003 ×3：电机侧 12V，逻辑共地
PMS5003：5V（勿从 ESP 3.3V 供电）
SHT31 / SCD30：3.3V


六、从 Waveshare LCD-4 迁到 4848 必做
------------------------------------
[必] 硬件包改为 guition-esp32-s3-4848s040.yaml，去掉 waveshare_io_ch32v003
[必] I2C 改接到 GPIO19/45（不再用 15/7）
[必] 风机 PWM/FG 改接 GPIO35/36/37/33（不再用 4/6/0/16）
[可] PMS UART 仍用 43/44，接板载 UART 座
[可] esphome run i2c-scan.yaml 确认 MCP 地址


附录：Waveshare ESP32-S3-Touch-LCD-4 引脚（历史）
----------------------------------------------
I2C 15/7；风机 4/6/0/16；PMS 43/44；MCP 常 0x21(V3) 或 0x20(V4+CH32@0x24)
详见 packages/pinmap_waveshare_lcd4.yaml
